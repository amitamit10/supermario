# תקציר

---

פרויקט זה הוא **משחק פלטפורמה** בסגנון הקלאסי "סופר מריו", הממומש בשפת C# תחת מסגרת ה-WinForms של .NET. המשחק כולל מנגנוני פיזיקה מדויקים, שישה סוגי אויבים, שלושה שלבים עיצוביים ידניים ושניים נוצרים אוטומטית, פריטי כוח-על ומערכת ניקוד.

הייחוד הבולט בפרויקט הוא **רכיב הבינה המלאכותית**: סוכן בשם לואיג'י לומד לשחק במשחק בעצמו ללא תכנות ידני של הכללים. הלמידה מתבצעת באמצעות **Neuroevolution** — שילוב של **רשת נוירונים** ו**אלגוריתם גנטי**. אוכלוסייה של 60 סוכנים רצים בו-זמנית, כאשר הסוכנים החזקים ביותר מעבירים את ה"גנים" שלהם לדור הבא. לאחר מספר דורות הסוכן לומד לקפץ על מכשולים, לחצות פלטפורמות ולהגיע למרחקים גדולים יותר ויותר.

**תוצאות עיקריות:**

- משחק פלטפורמה פונקציונלי ומהנה עם תחושת תנועה אותנטית
- מנוע פיזיקה עצמאי עם תנגשויות, כוח משיכה וקפיצות משתנות
- סוכן לואיג'י המסוגל ללמוד לשחק מאפס ללא הנחיות מפורשות
- ממשק אימון חי המציג את הדור, הניקוד, וויזואליזציה של מבנה הרשת

**שפות וכלים:** C#, .NET Framework 4.7.2, WinForms, GDI+, Python (לייצור sprites).
