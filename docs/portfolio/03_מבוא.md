## 1.1 רקע

משחקי פלטפורמה הם אחד הז'אנרים הותיקים בתולדות תעשיית המשחקים. **Super Mario Bros** של Nintendo (1985) הגדיר את התבנית: שחקן הקופץ על פלטפורמות, נמנע מאויבים, ואוסף פריטים לאורך שלבים. מאחורי הפשטות הכאילו-פעוטה מסתתר מנגנון פיזיקה מדויק — כוח משיכה, תאוצה, עצירה הדרגתית, קפיצה עם גובה משתנה לפי אורך הלחיצה.

בפרויקט זה בחרתי לבנות מחדש את חוויית המשחק הקלאסי הזו בעצמי, תוך שאני מאמת מתמטית כל קבוע פיזיקלי ובונה ארכיטקטורה נקייה. בנוסף, הרחבתי את הפרויקט בכיוון שלא קיים במשחק המקורי: **בינה מלאכותית** שלומדת לשחק מאפס.

## 1.2 מוטיבציה

שלוש שאלות הניעו את הפרויקט:

1. **כיצד מיישמים פיזיקה אמינה של פלטפורמה?** — לא להשתמש בספרייה מוכנה, אלא לבנות בעצמי מנוע המסוגל לנהל קפיצות, תאוצה, התנגשויות ומוות.

2. **כיצד מפתחים משחק ברמה מקצועית ב-C# ללא מנוע משחקים?** — להשתמש ב-WinForms וב-GDI+ בלבד, תוך ניהול קצב פריימים, עיבוד ספרייט-שיטים ואנימציה.

3. **האם ניתן ללמד מחשב לשחק ללא תכנות ידני של הכללים?** — לממש Neuroevolution: הגדיר לסוכן קלטים חישתיים ומטרה, ותן לאלגוריתם הגנטי למצוא את המשקולות הנכונות.

## 1.3 מטרות הפרויקט

| מטרה | תיאור |
|------|--------|
| **פונקציונלית** | בנות משחק פלטפורמה ניתן למשחק עם שלבים, אויבים ופריטים |
| **הנדסית** | מנוע פיזיקה עצמאי, ארכיטקטורה מודולרית, ביצועים טובים |
| **בינה מלאכותית** | סוכן לומד שמשפר ביצועיו דור אחר דור ללא התערבות ידנית |
| **ויזואלית** | ספרייט-שיטים אותנטיים, אנימציות, תפריט מונפש |

## 1.4 נושאים שנלמדו במסגרת הפרויקט

### מתכנות

- תכנות מונחה-עצמים (OOP) מתקדם: מחלקות חלקיות (`partial class`), ממשקים, ירושה
- ניהול ריבוי אירועים (event-driven programming) ב-WinForms
- ניהול גרפיקה עם GDI+ ו-`Graphics`
- טיפול בזיכרון: `Dispose`, `using`, הימנעות מדליפות
- ביצועים: `SuspendLayout`/`ResumeLayout`, קאש, צמצום redraws

### אלגוריתמים ומתמטיקה

- פתרון התנגשויות AABB (Axis-Aligned Bounding Box)
- פיזיקה: אינטגרציה מספרית, כוח משיכה, תאוצה
- רשתות נוירונים: Forward Pass, `tanh` activation
- אלגוריתם גנטי: elitism, crossover, mutation

### כלים ותהליכי פיתוח

- ניהול גרסאות עם Git (ענפים, PR, merge)
- Python לייצור נכסים (pixel art, sprite sheets)

## 1.5 מגבלות הפרויקט

- המשחק רץ על **Windows בלבד** (WinForms)
- אין שמירת התקדמות בין הפעלות
- הבינה המלאכותית לומדת בסביבת אימון פשוטה (ללא אויבים, ללא מטבעות)
- לא מיושמת מוסיקת רקע (רק גרפיקה)
