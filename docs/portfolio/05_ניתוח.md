## 3.2 מודל נתונים

### ישויות מרכזיות

```
Player (שחקן)
├── Health: int = 3
├── Score: int
├── IsGrounded: bool
├── IsSuper: bool (גדול/קטן)
├── Position: Point
├── HorizontalVelocity: float
└── VerticalVelocity: float

Enemy (אויב — בסיס משותף)
├── Position: Point
├── IsAlive: bool
├── Direction: int (-1 / +1)
└── [טיפוס-ספציפי]

GameObjectS (פלטפורמה/צינור)
├── Visual: PictureBox
├── Bounds: Rectangle (world-space)
└── IsGround: bool

GameData — GameObjects (פריטי משחק)
├── Mushroom
│   ├── Position, IsActive
│   └── VelocityX, VelocityY
├── Coin
│   ├── Position, IsActive
│   └── AnimFrame
└── QuestionBlock
    ├── Position, IsActivated
    └── PowerUpType (Coin / Mushroom)
```

### מודל הבינה המלאכותית

```
NetParams (הגדרות גלובליות)
├── PopulationSize: int = 60
├── MutationRate: float = 0.05
├── SurviveRate: float = 0.30
└── NetworkShape: int[] = {4, 6, 4, 2}

NeuralNetwork (מוח של סוכן)
├── Layers: Layer[]
└── Forward(inputs: float[]) → float[]

MarioAgent (סוכן לואיג'י)
├── Brain: NeuralNetwork
├── Position: Point
├── IsAlive: bool
├── Fitness: float (= מקסימום X שהושג)
└── ComputeInputs(world) → float[4]

Population (אוכלוסייה)
├── Agents: MarioAgent[]
├── Generation: int
└── CreateNewGeneration() — elitism + crossover + mutation
```

## 3.3 מודל הפיזיקה

### קבועים פיזיקליים

| קבוע | ערך | יחידות | תפקיד |
|------|-----|---------|--------|
| `MoveSpeed` | 4.4 | פיקסל/tick | מהירות יעד בריצה |
| `MaxMoveSpeed` | 5.6 | פיקסל/tick | תקרת מהירות אופקית |
| `GroundAccel` | 0.75 | פיקסל/tick² | תאוצה על הקרקע |
| `AirAccel` | 0.42 | פיקסל/tick² | תאוצה באוויר |
| `GroundDecel` | 0.55 | פיקסל/tick² | עצירה על הקרקע |
| `AirDecel` | 0.16 | פיקסל/tick² | עצירה באוויר |
| `Gravity` | 0.58 | פיקסל/tick² | כוח משיכה |
| `JumpPower` | -13.8 | פיקסל/tick | עוצמת קפיצה (שלילי = למעלה) |
| `MaxFallSpeed` | 15.5 | פיקסל/tick | מהירות נפילה מקסימלית |
| `JumpReleaseGravity` | ×2.4 | — | מכפיל כוח משיכה בשחרור מקש |

> **הערה**: tick = 16ms, כלומר ~60 ticks לשנייה.

### אלגוריתם אינטגרציה (בכל tick)

```
1. חשב horizontalVelocity ← Approach(currentVX, target, accel)
2. קצץ ל-[-MaxMoveSpeed, +MaxMoveSpeed]
3. הזז: preciseX += horizontalVelocity
4. גבול עולם: clamp X ∈ [0, 2950]
5. אם קפיצה נלחצה AND IsGrounded: VerticalVelocity = -13.8
6. אם באוויר: VerticalVelocity += gravity (× 2.4 אם קפיצה שוחררה)
7. קצץ: VerticalVelocity ≤ 15.5
8. הזז: preciseY += VerticalVelocity
9. עגל: Position = (round(preciseX), round(preciseY))
```

### אלגוריתם התנגשות (AABB — ResolveSmallestOverlap)

```
נתון: גוף נע (body) חופף פלטפורמה (plat)

1. חשב 4 חדירות:
   ot = body.Bottom - plat.Top        (חדירה מלמעלה)
   ob = plat.Bottom - body.Top        (חדירה מלמטה)
   ol = body.Right  - plat.Left       (חדירה משמאל)
   or = plat.Right  - body.Left       (חדירה מימין)

2. בחר מינימום מבין הארבע

3. אם מינימום == ot AND ot < 30px:
       → נחיתה: SnapAbove(plat.Top), VY = 0, IsGrounded = true
   אם מינימום == ob:
       → פגיעה בתקרה: SnapBelow(plat.Bottom), VY = 0
   אם מינימום == ol:
       → חסימה מצד שמאל
   אם מינימום == or:
       → חסימה מצד ימין
```

> **ספירה**: סף 30px (לא 25px) מבטיח שגוף הנופל ב-15.5px/tick לא יחמיק דרך הרצפה.

## 3.4 מודל הבינה המלאכותית

### ארכיטקטורת רשת הנוירונים

```
קלט (4 נוירונים)         שכבה נסתרת 1    שכבה נסתרת 2    פלט (2 נוירונים)
────────────────────      ──────────────   ──────────────   ──────────────────
I1: מרחק לחריץ הבא   ──►│              │──►│            │──►  O1: כיוון (-1/+1)
I2: מרחק לאויב       ──►│  6 נוירונים  │──►│  4 נוירונים│──►  O2: קפיצה (bool)
I3: הפרש גובה פלטפורמה►│              │   │            │
I4: האם על הקרקע      ──►│              │   │            │
```

כל נוירון מחשב: `output = tanh(Σ(input_i × weight_i) + bias)`

### אלגוריתם גנטי

```
דור 1:
  60 רשתות עם משקולות אקראיות

בכל דור:
  1. כל סוכן חי רץ עד שנופל / נתקע / מגיע לסוף
  2. fitness(סוכן) = max(X) שהשיג
  3. מיין סוכנים לפי fitness
  4. שמור 30% עליונים (elitism)
  5. מלא את שאר האוכלוסייה:
       א. בחר אב ואם מהשורדים
       ב. Crossover: לכל משקולה בחר מאיזה הורה
       ג. Mutation: בהסתברות 5% — שנה משקולה אקראית
  6. חזור

קריטריון עצירה: לא אוטומטי — המשתמש עוצר ידנית
```
