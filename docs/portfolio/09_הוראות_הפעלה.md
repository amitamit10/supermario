## 7.2 התקנה והפעלה

### שלב 1 — הורדת הפרויקט

```
1. הורד את קובץ ה-ZIP של הפרויקט
2. חלץ לתיקייה כלשהי (לדוגמה: C:\Games\supermario)
```

### שלב 2 — בנייה (Build)

```
1. פתח את supermario.sln ב-Visual Studio
2. בחר Build > Build Solution (Ctrl+Shift+B)
3. ודא שאין שגיאות בחלונית Output
```

### שלב 3 — הפעלה

```
1. לחץ F5 (Run with debugging) או Ctrl+F5 (Run without debugging)
2. המשחק יפתח בחלון ללא מסגרת (borderless)
```

> **הערה**: תיקיית `assets/textures/sprite_sheets/` חייבת להיות באותה תיקייה כמו קובץ ה-.exe. אם חסרה — המשחק עדיין יעבוד אך יצייר גרפיקה פרוצדורלית (GDI+) במקום ספרייט-שיטים.

## 7.5 מכניקות המשחק

### חיים וניקוד

- מתחילים עם **3 חיים**
- כל פגיעת אויב: **-1 HP**
- כשה-HP מגיע ל-0: **מוות** — חיים-- ואתחול רמה
- כשהחיים נגמרים: **Game Over** + חזרה לתפריט
- דריסת אויב: **+100 נקודות**
- איסוף מטבע: **+1 למונה המטבעות**

### כוח-על (Super)

1. מצא בלוק שאלה (? block) — ניתן לזהות לפי הצבע הצהוב
2. קפוץ **מלמטה** כדי להפעיל — תצוא פטרייה
3. גע בפטרייה — מריו יגדל (Super Mario)
4. בסופר: פגיעת אויב גורמת לכיווץ חזרה, **ללא אובדן HP**

### ממשק האימון

```
┌─────────────────────────────┬────────────────────┐
│                             │  LUIGI AI TRAINER  │
│   Canvas — אזור האימון      │                    │
│                             │  GENERATION: 14    │
│   60 לואיג'ים רצים          │  ALIVE: 23         │
│   (עם טבעת זהב על המוביל)   │  BEST: 1840        │
│                             │  ALL-TIME: 2100    │
│                             │                    │
│                             │  [NN Visualizer]   │
│                             │                    │
│                             │  SETTINGS          │
│                             │  Population: 60    │
│                             │  Mutation: 5%      │
│                             │  Survive: 30%      │
│                             │  Shape: 4,6,4,2    │
│                             │                    │
│                             │  [APPLY]           │
│                             │  [▶ START]         │
│                             │  [⟳ RESET]         │
│                             │  [← BACK]          │
└─────────────────────────────┴────────────────────┘
```

### כפתורי הבקרה

| כפתור | פעולה |
|-------|--------|
| **▶ START** | מתחיל את הסימולציה |
| **⏸ PAUSE** | משהה (אותו כפתור) |
| **⟳ RESET** | אתחול מחדש — דור 0, רשתות אקראיות |
| **APPLY** | מחיל הגדרות חדשות ומאתחל |
| **← BACK** | חזרה לתפריט הראשי |
| **Escape** | זהה ל-BACK |

### הגדרות הניתנות לשינוי

| הגדרה | טווח | ברירת מחדל | השפעה |
|-------|------|-----------|--------|
| **Population** | 10 – 500 | 60 | גודל האוכלוסייה בכל דור |
| **Mutation %** | 1 – 50 | 5 | הסתברות לשינוי משקולה |
| **Survive %** | 5 – 80 | 30 | אחוז הסוכנים ששורדים לדור הבא |
| **Net Shape** | מחרוזת | `4,6,4,2` | מבנה שכבות הרשת — **חייב להתחיל ב-4 ולסיים ב-≥2** |

> **אזהרה**: שינוי ה-Net Shape לערך לא תואם (לדוגמה: `3,6,2`) יגרום להודעת שגיאה. הספרה הראשונה **חייבת להיות 4** (מספר הקלטים) והאחרונה **חייבת להיות לפחות 2** (מספר הפלטים).

### קריאת הויזואליזציה

- **נקודות** = נוירונים. צהוב בהיר = אקטיבציה גבוהה
- **קווים כחולים** = משקולה חיובית (מחזקת)
- **קווים אדומים** = משקולה שלילית (מעכבת)
- **עובי הקו** = גודל המשקולה המוחלט

### כיצד לדעת שהסוכן לומד?

- ה-**BEST** ו-**ALL-TIME BEST** צריכים לעלות לאורך הדורות
- בדורות הראשונים — רוב הלואיג'ים נופלים לבור מיד
- אחרי 15-30 דורות — חלק מהם מצליחים לחצות מספר פלטפורמות
- מוביל בולט = לואיג'י עם **טבעת זהב** שהמצלמה עוקבת אחריו
