## הפקת PDF

כדי להפיק PDF מהתיק, ניתן להשתמש באחת מהדרכים הבאות:

### אפשרות א' — Pandoc (מומלץ)
```bash
# התקנה
winget install JohnMacFarlane.Pandoc

# הפקת PDF לכל הקבצים
pandoc docs/portfolio/01_שער.md docs/portfolio/02_תקציר.md \
       docs/portfolio/03_מבוא.md docs/portfolio/04_הגדרת_הבעיה.md \
       docs/portfolio/05_ניתוח.md docs/portfolio/06_תיכון.md \
       docs/portfolio/07_מימוש.md docs/portfolio/08_בדיקות.md \
       docs/portfolio/09_הוראות_הפעלה.md docs/portfolio/10_סיכום.md \
       docs/portfolio/11_ביבליוגרפיה.md \
       -o תיק_פרויקט_סופר_מריו.pdf \
       --pdf-engine=xelatex -V lang=he -V mainfont="Arial"
```

### אפשרות ב' — VS Code + Markdown PDF extension
1. פתח VS Code
2. התקן את extension "Markdown PDF"
3. פתח כל קובץ .md ולחץ Ctrl+Shift+P → "Markdown PDF: Export (pdf)"

### אפשרות ג' — GitHub / GitLab
העלה לריפו ב-GitHub — הקבצים מוצגים כ-Markdown מעוצב בדפדפן, ניתן להדפיס לPDF דרך הדפדפן (Ctrl+P → Save as PDF).
