## 5.3 מנגנון הפיזיקה — קטעי קוד מרכזיים

### פונקציית Approach (תאוצה חלקה)

```csharp
static float Approach(float current, float target, float rate)
{
    if (current < target) return Math.Min(current + rate, target);
    if (current > target) return Math.Max(current - rate, target);
    return current;
}
```

פונקציה זו מייצרת תנועה חלקה: הגוף מתקרב למהירות היעד בקצב קבוע, ללא חריגה.

### תאוצה אופקית (תוך Player.Move)

```csharp
float target = direction * MoveSpeed;
float accel  = IsGrounded ? GroundAcceleration : AirAcceleration;
float decel  = IsGrounded ? GroundDeceleration : AirDeceleration;

horizontalVelocity = (direction != 0)
    ? Approach(horizontalVelocity, target, accel)
    : Approach(horizontalVelocity, 0f,     decel);

horizontalVelocity = Math.Max(-MaxMoveSpeed,
                     Math.Min( MaxMoveSpeed, horizontalVelocity));
preciseX += horizontalVelocity;
```

### קפיצה משתנה

```csharp
if (jumpPressed && IsGrounded)
{
    VerticalVelocity = JumpPower;   // -13.8

if (!IsGrounded)
{
```

### פתרון התנגשות (ResolveSmallestOverlap)

```csharp
float ot  = body.Bottom - plat.Top;
float ob  = plat.Bottom - body.Top;
float ol  = body.Right  - plat.Left;
float orr = plat.Right  - body.Left;

float min = Math.Min(Math.Min(ot, ob), Math.Min(ol, orr));

if      (min == ot  && ot < 30)  player.LandOn(plat.Top - bodyHeight);
else if (min == ob)              player.HitCeiling(plat.Bottom);
else if (min == ol)              player.BlockHorizontal(plat.Left - bodyWidth);
else if (min == orr)             player.BlockHorizontal(plat.Right);
```

## 5.5 מנגנון הבינה המלאכותית

### Forward Pass של רשת נוירונים

```csharp
// NeuralNetwork.cs
public float[] Forward(float[] inputs)
{
    float[] current = inputs;
    foreach (var layer in Layers)
        current = layer.Forward(current);
    return current;
}

// Layer.cs
public float[] Forward(float[] inputs)
{
    float[] outputs = new float[Neurons.Length];
    for (int i = 0; i < Neurons.Length; i++)
        outputs[i] = Neurons[i].Forward(inputs);
    return outputs;
}

// Neuron.cs
public float Forward(float[] inputs)
{
    float sum = Bias;
    for (int i = 0; i < Weights.Length; i++)
        sum += Weights[i] * inputs[i];
    return (float)Math.Tanh(sum);
}
```

### יצירת דור חדש (Neuroevolution)

```csharp
// Population.CreateNewGeneration()
var sorted = Agents.OrderByDescending(a => a.Fitness).ToArray();
int survivors = (int)(sorted.Length * SurviveRate);

var nextGen = new MarioAgent[Agents.Length];

// שמור את הטוב ביותר ישירות (elitism)
nextGen[0] = sorted[0].Clone();

for (int i = 1; i < nextGen.Length; i++)
{
    var parent1 = sorted[_rng.Next(survivors)];
    var parent2 = sorted[_rng.Next(survivors)];
    nextGen[i]  = Crossover(parent1, parent2);
    Mutate(nextGen[i]);
}
Agents = nextGen;
Generation++;
```

### Crossover בין שני הורים

```csharp
MarioAgent Crossover(MarioAgent p1, MarioAgent p2)
{
    var child = p1.Clone();
    var w1 = p1.Brain.GetWeights();
    var w2 = p2.Brain.GetWeights();
    var wc = new float[w1.Length];

    for (int i = 0; i < wc.Length; i++)
        wc[i] = (_rng.NextDouble() < 0.5) ? w1[i] : w2[i];

    child.Brain.SetWeights(wc);
    return child;
}
```

### מוטציה

```csharp
void Mutate(MarioAgent agent)
{
    var weights = agent.Brain.GetWeights();
    for (int i = 0; i < weights.Length; i++)
        if (_rng.NextDouble() < MutationRate)
            weights[i] += (float)(_rng.NextGaussian() * 0.3);
    agent.Brain.SetWeights(weights);
}
```
