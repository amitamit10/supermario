## 6.1 אסטרטגיית הבדיקות

הבדיקות בפרויקט זה הן **בדיקות ידניות** (manual testing), מאחר שאין ב-WinForms תשתית בדיקות אוטומטיות מובנית. הבדיקות מחולקות ל:

1. **בדיקות יחידה (Unit)** — בדיקת פונקציות פיזיקה ומתמטיקה בבידוד
2. **בדיקות אינטגרציה** — בדיקת אינטראקציות בין מודולים
3. **בדיקות מערכת** — שחקן שמשחק את השלבים מתחילה לסוף
4. **בדיקות קצה (Edge Cases)** — מצבים קיצוניים שגרמו לבאגים

## 6.7 באגים שנמצאו ותוקנו

| באג | תיאור | תיקון |
|-----|-------|-------|
| מנהרה בנפילה | שחקן עבר דרך הרצפה בנפילה מהירה | סף נחיתה הועלה מ-25 ל-30px (`1e82bb3`) |
| קפיצה אוטומטית | החזקת Space דרך נחיתה הפעילה קפיצה שנית | `jumpEdge` — רק על rising edge (`b67a336`) |
| אויבים בלתי נראים | אויבים נשלחו מאחורי הרצפה | הסרת `SendToBack()` מפונקציות spawn (`be1f398`) |
| ריסוק בקפיצה | ריסוק אויב כשנוגעים מהצד בגובה הנכון | הוספת גרד `VY ≥ 0` לריסוק (`95a0a36`) |
| גולם כוח נפילה | נזק נפילה אחרי כל קפיצה רגילה | סף הועלה מ-60→120→220 px (`5a8c95c`, `3cdb3fe`) |
| גלישת int אנימציה | `globalTick` הגיע לערך מקסימלי אחרי שעות | `globalTick % 168` (LCM של כל מחזורי אנימציה) (`1e82bb3`) |
| דליפת זיכרון HUD | fonts ו-TextureLoader לא שוחררו | Dispose נוסף בכל המקומות (`3cdb3fe`) |
| JumpingEnemy קצב חצי | קפץ פעם בכל 3 שנ' במקום 1.5 | תיקון timer logic (`d208cb7`) |
| רשת AI שגויה | IndexOutOfRange כשה-shape לא תואמת 4 כניסות | ולידציה בטעינת קובץ ובשינוי שדה Shape (`AI commit`) |
